一句话简介：一个用于浏览 Spark Store 与 APM Store 的终端 TUI 工具。

详细简介：
Spark Store TUI 是一个基于 Bash、fzf、curl、jq、aria2c 的终端应用商店浏览工具。它支持浏览 Spark Store / APM Store 元数据、查看应用详情、预览图片，并使用 Metalink + aria2c 下载应用包。Debian-like 系统默认使用 Spark Store，非 Debian-like 系统默认使用 APM Store，避免在非 Debian 系统中展示不适合安装的 Spark 内容。
